# Writing Buddy

A local web editor for Obsidian markdown files with scene awareness, story cards, and writing analytics.

## Quick Start

1. Install Python 3.11+
2. Run `pip install fastapi uvicorn`
3. Run `run.bat` (Windows) or `run.sh` (Mac/Linux)
4. Open http://localhost:8000 in Chrome
5. Configure your Obsidian vault path on first run

## Features

- Direct read/write to Obsidian vault files
- Scene splitting with visual dividers
- Word counts per scene, file, and folder
- Story cards (Phase 2) and writing analytics (Phase 3)

See [PLAN.md](PLAN.md) for full architecture.
